# PORTFOLIO-PLATFORM
A modern, responsive, and visually appealing Portfolio Website built to showcase personal projects, skills, and contact information. Designed with clean UI/UX principles and optimized for multiple devices.
# Features
- Fully Responsive Design (Mobile + Desktop)
- About Me Section
- Project Showcase
-  Contact Section UI
-  mooth scrolling & clean layout
-  Modern UI/UX design
  
# Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Styling: Custom CSS
- Version Control: Git & GitHub

  # Project Structure
 ```
Portfolio-Platform/
│── public/              # Images & assets
│── styles/              # CSS files
│── script.js            # JavaScript logic
│── index.html           # Main entry point
│── package.json         # Project metadata
│── tsconfig.json        # TypeScript config

```
